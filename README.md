# 🐾 AnimalWorld

Página web de animales construida con **React + TypeScript + Tailwind CSS v4 + Vite**.

## APIs gratuitas integradas

| API | URL | Uso |
|-----|-----|-----|
| 🐶 TheDogAPI | https://thedogapi.com | Razas e imágenes de perros |
| 🐱 TheCatAPI | https://thecatapi.com | Razas e imágenes de gatos |

## Instalación local

```bash
npm install
npm run dev
```

## Build para producción

```bash
npm run build
```

## Subir a GitHub Pages

```bash
# 1. Instala gh-pages
npm install --save-dev gh-pages

# 2. Agrega en package.json:
# "homepage": "https://TU_USUARIO.github.io/animal_world",
# "predeploy": "npm run build",
# "deploy": "gh-pages -d dist"

# 3. Despliega
npm run deploy
```

## Secciones

- 🦁 **Hero** – Pantalla de bienvenida animada
- 🐶 **Perros** – Buscador de razas con TheDogAPI
- 🐱 **Gatos** – Buscador de razas con TheCatAPI
- 🧠 **Curiosidades** – Datos fascinantes sobre animales
- 📸 **Galería** – Fotos aleatorias en tiempo real
